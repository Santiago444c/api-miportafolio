# API CRUD de Usuarios - FastAPI + MySQL

Proyecto backend desarrollado con FastAPI y MySQL que implementa un CRUD básico de usuarios.

## Descripción

Esta API permite:

- Crear usuarios
- Listar usuarios
- Actualizar usuarios
- Eliminar usuarios

El proyecto demuestra:

- Arquitectura modular
- Conexión a base de datos con SQLAlchemy
- Validación con Pydantic
- Separación de responsabilidades

---

## Tecnologías usadas

- Python 3.11
- FastAPI
- MySQL
- SQLAlchemy
- Pydantic
- Uvicorn
